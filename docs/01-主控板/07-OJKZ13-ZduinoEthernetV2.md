# Zduino Ethernet V2（Arduino Ethernet兼容控制器）

## 概述

Zduino Ethernet是基于Arduino Ethernet的微控制器板，我们在官方原作基础上，集成了USB下载、POE供电、TF插槽等功能，并且完全兼容UNO引脚位置。使用它，你可以快速的将你的控制器接入互联网从而搭建你的网络应用。
