# Arduino® Nano R4 产品说明

## 概述
Nano R4 将 Arduino 标志性的易用性与强大的瑞萨 RA4M1 微控制器相结合，让开发者能够轻松从 UNO R4 原型设计过渡到生产级应用。

这款紧凑型多功能开发板：
- 延续 Nano 系列经典引脚布局
- 采用半孔设计便于硬件集成
- 板载 Qwiic 连接器可快速扩展传感器与外设
- 可编程 RGB LED 能直观反馈系统状态
- 单面元件设计适合空间受限产品
- 便于在定制载板上进行表面贴装焊接

> 兼容 Arduino 常用工具和库（包括 Arduino UNO R4 Minima 的生态），为开发者提供构建紧凑型、可联网、生产级解决方案的捷径。

## 核心优势
### 从原型到量产的无缝过渡
- 沿用 UNO R4 系列已验证的 RA4M1 微控制器
- 代码几乎无需修改

### 紧凑的生产级设计
- 4.3 x 1.7 厘米迷你尺寸
- 半孔引脚与单面元件布局
- 轻松集成至定制硬件

### 便捷扩展能力
- 内置 Qwiic 连接器
- 额外 5V I2C 接口
- 无缝连接各类传感器、执行器及 Modulino 模块

### 可定制化反馈
- 通过可编程 RGB LED 指示系统状态
- 加速调试并增强用户交互体验

### 灵活熟悉的平台
- 享有 Arduino 丰富的生态体系
- 直观的开发环境
- 强大的社区支持

## 应用场景
### 嵌入式工业系统
- 空间有限但可靠性要求严苛
- 生产级特性保障稳定性

### 智能家居
- 快速开发联网智能产品
- 便捷集成传感器与执行器
