# MG995 金属齿轮舵机产品资料 & Arduino教程

## 一、MG995舵机产品资料

### 规格参数
| 项目               | 参数值                  |
|--------------------|-------------------------|
| 工作电压           | 4.8V-7.2V (推荐6V)      |
| 空载电流           | 10mA (@6V)              |
| 堵转电流           | 800mA (@6V)             |
| 工作温度           | 0°C - 55°C              |
| 扭矩               | 9.4kg·cm (@4.8V)        |

| 线序               | 棕色(GND)/红色(VCC)/橙色(Signal) |

### 特点
- 360°连续旋转（需修改内部限位）
- 双滚珠轴承设计
- 大扭矩输出
- 内置过温保护电路

## 二、Arduino连接教程

### 所需材料
- Arduino UNO开发板
- MG995舵机 ×1
- 外部电源（推荐6V/2A）
- 1000μF电容（可选，用于稳压）
- 面包板及杜邦线

### 电路连接图
MG995 Arduino

棕色线 → GND
红色线 → 外部电源正极（需与Arduino共地）
橙色线 → D9（PWM引脚）

外部电源负极 → Arduino GND
（建议在电源正负极并联1000μF电容）

### 基础控制代码
```arduino
#include <Servo.h>

Servo mg995;  // 创建舵机对象

void setup() {
  mg995.attach(9);  // 连接D9引脚
  // mg995.attach(9, 500, 2500);  // 如需自定义脉宽范围
}

void loop() {
  mg995.write(0);    // 0度位置
  delay(1000);
  mg995.write(90);   // 90度中位
  delay(1000);
  mg995.write(180);  // 180度位置
  delay(1000);
}
高级使用技巧
平滑运动控制：

arduino
for(int pos=0; pos<=180; pos+=1){
  mg995.write(pos);
  delay(15);  // 控制运动速度
}
```

外部电源注意事项：

必须与Arduino共地

建议使用稳压模块

大电流负载时单独供电

常见问题解决：

抖动问题：增加滤波电容

力度不足：检查电源电流是否达标

发热严重：避免长时间堵转
